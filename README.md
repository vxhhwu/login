## About the Project

This is a simple desktop application created for practicing **PyQt5** skills. It provides a basic authentication system with a graphical user interface.

The app allows users to:
- **Log in** with an existing username and password.
- **Register** a new account.
- **Continue as a guest** without an account.

It serves as a practical exercise for learning how to handle windows, dialogs, input validation, and JSON data storage in PyQt5.

## Features

- ✅ User-friendly login form (username and password fields).
- ✅ Registration dialog to create new credentials.
- ✅ Guest login option.
- ✅ Persistent storage of user credentials using a local `base.json` file.
- ✅ Basic error handling (incorrect password, user not found).

## Technology Stack

- **Language:** Python 100%
- **Framework:** PyQt5 for the graphical interface.
- **Data Storage:** JSON file (`base.json`).
